Detailed Description

virtual Class for all the roles.

An Role object represents a player during the game. It provides methods to manage interactions with other players.

Card * Role::chooseCard ( gameState_t *  state,
Deck *  hand,
int  brainLevel 
) [static]

Choose which Card to play, using different algorithms.

This function chooses which card to play, using different levels of complexity. The main concepts are of the algorithms are:

  • BASE: play lowest card
  • FINAL RUSH: play highest card
  • SAVE: do not play

Every option generated by an algorithms has a level, based on:

  • value of the Card object
  • count of the Card object
  • level of the Card object
  • Cards in the Deck
  • algorithm relevance

This method is declared as static so that it can be used in every situation (e.g.: advice to user)

hand player's Deck
